## Installation
```bash
yarn global add webgif || npm i -g webgif
```

## Usage
To navigate to `https://giphy.com/search/lol` and make an animated GIF of duration `60` seconds, execute:
```bash
webgif https://giphy.com/search/lol -d 60
```
